# Plugins officiels

Ce dossier hébergera les plugins officiels d'Explorer Engine (un sous-dossier par plugin), chacun dépendant **uniquement** de `@explorer-engine/plugin-sdk` (jamais des internes du core, ni d'un autre plugin — voir ENGINE_CONSTITUTION L15, chapitre 10 §10.6bis).

- **Statut** : conteneur vide (P0-T1). Aucun plugin n'est créé à ce stade.
- **Référence** : chapitre 10, ADR-006.
