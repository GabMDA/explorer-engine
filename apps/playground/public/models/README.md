# P2-T2 model fixture

`cube.glb` — a minimal, self-contained, **uncompressed** glTF 2.0 binary used to
exercise the Model Loader (roadmap P2-T2). It is a unit cube (half-size 0.5,
centred at the origin) with `POSITION` + `NORMAL` attributes, indexed triangles,
and a single PBR metallic-roughness material (base colour ≈ `#3ba7ff`).

- No Draco, KTX2 or Meshopt compression.
- No external textures or any secondary network dependency.
- ~1.5 KB.

## Provenance

Generated by a small dependency-free packer (no runtime dependency, no
conversion tool): it builds the 6 cube faces in code, writes the interleaved
`POSITION`/`NORMAL`/index buffers, and assembles the GLB container (12-byte
header + JSON chunk + BIN chunk) by hand. The generator is not part of the
shipped engine; the committed `cube.glb` is the reproducible output.

The byte-identical content is also inlined (base64) in
`packages/renderer-three/src/model-loader.real.test.ts` so the real
`GLTFLoader.parse` path is covered without a filesystem dependency.
